Example 4.1.5. The Fibonacci Sequence.

The Fibonacci sequence \(F_0, F_1, F_2, \dots\) is defined by the recurrence:
\begin{equation*} F_n = F_{n-1} + F_{n-2} \text{ For } n \ge 2, \text{ where } F_0 = 0, F_1 = 1 \end{equation*}

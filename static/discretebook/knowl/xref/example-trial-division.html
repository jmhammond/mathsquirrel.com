Example 3.2.8.

Use trial division to determine if any of the following are prime:
  1. 127
  2. 143
  3. 54323
Solution.
  1. Since, \(\sqrt{127} \approx 11\text{,}\) we need only test primes up through 11. Noticing that none of 2, 3, 5, 7, or 11 divide evenly, we conclude 127 is prime.
  2. Here, \(\sqrt{143} \approx 12\) so again we test only up through 11, the largest prime under 12.. We check that 2, 3, 5, 7 do not work but \(143 = 11\cdot 13\text{,}\) so we see that \(143\) is not a prime number.
